# Project Requirements

## Completed Features
- [x] Global MRN (Medical Record Number) Search.
- [x] Doctor Consultation Desk (Unified Clinical Workspace).
- [x] Nursing Console with Vitals & MAR.
- [x] Bed Management & IPD Workflow.
- [x] Pharmacy, Lab, Radiology Dashboards.
- [x] Quality Analytics Hub & Medical Records Dashboard.
- [x] Patient Registration with Automatic MRN Generation.

## Current Priorities (Based on User Feedback)
- [ ] **MRN Consistency**: Ensure MRN is visible and searchable in every patient-related view.
- [ ] **Deduplication**: Merge similar dashboards (ER, IPD, OPD) into a unified workspace structure.
- [ ] **Clinical Depth**: Add SOAP notes, Order Sets, and advanced clinical charting.
- [ ] **System Robustness**: Fix identified UI/UX gaps and functional errors.
- [ ] **Enterprise UX**: High-density layouts for professional productivity.

## Business Rules
- Every patient must have a unique MRN.
- Access to clinical data is role-based (Doctor vs. Nurse vs. Staff).
- Real-time synchronization for clinical events.
