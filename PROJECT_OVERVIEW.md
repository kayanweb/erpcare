# Hospital Information System (HIS) - Project Overview

## Goal
Build a complete, production-ready, enterprise-grade Hospital Information System (HIS) with integrated EMR, ERP, and Patient Engagement modules.

## Core Modules
1. **Front Office & ADT**: Patient registration, appointments, bed management.
2. **Clinical Workspaces**: Doctor Desk, Nursing Console, EMR, ER, IPD, OPD.
3. **Ancillary Services**: Pharmacy, Laboratory, Radiology, Blood Bank.
4. **Operations & Finance**: Billing, HR, Finance, TPA Management.
5. **Quality & Analytics**: Performance dashboards, clinical gaps, medical records.

## Visual Identity
- **Enterprise Modern**: Clean, high-density, professional UI.
- **Color Palette**: Indigo/Slate for core, specialized colors for departments (ER: Rose, Pharmacy: Emerald, etc.).
- **Typography**: Inter for UI, JetBrains Mono for clinical data (vitals, MRN).
